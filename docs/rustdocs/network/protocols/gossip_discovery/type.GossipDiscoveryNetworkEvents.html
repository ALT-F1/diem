[−][src]Type Definition network::protocols::gossip_discovery::GossipDiscoveryNetworkEvents

type GossipDiscoveryNetworkEvents = NetworkEvents<GossipDiscoveryMsg>;

The interface from Network to Discovery module.

DiscoveryNetworkEvents is a Stream of PeerManagerNotification where the raw Bytes rpc messages are deserialized into GossipDiscoveryMsg types. DiscoveryNetworkEvents is a thin wrapper around a channel::Receiver<PeerManagerNotification>.
