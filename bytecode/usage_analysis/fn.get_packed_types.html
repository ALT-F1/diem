pub fn get_packed_types(
    env: &GlobalEnv, 
    targets: &FunctionTargetsHolder
) -> BTreeSet<StructTag>

Get all closed types that may be packed by (1) genesis and (2) all transaction scripts. This makes some simplifying assumptions that are not correct in general, but hold for the current Libra Framework:

  • Transaction scripts have at most 1 type argument
  • The only values that can be bound to a transaction script type argument are Coin1, Coin2, and LBR. Passing any other values will lead to an aborted transaction. The first assumption is checked and will trigger an assert failure if violated. The second is unchecked, but would be a nice property for the prover.
